📖 Adjective Repetition as Adverbial Modifier

✔ haohao · manman · zaozao ✔ HSK 3 Grammar Point ✔ Based on Lessons 1-14

📘 Grammar Rule

📌 S + AA + (地) + V + O

Repeat an adjective, put it before the verb to describe how the action is done.

🔹 Common Repeated Adjectives (from lessons 1-14)

好好 (hǎohāo) 慢慢 (mànmān) 早早 (zǎozāo) 快快 (kuàikuāi) 轻轻 (qīngqīng) 大声 (dàshēng)

⭐ 你 好好 看看这本书。

📖 You read this book carefully.

⭐ 我 慢慢 吃饭。

🍚 I eat slowly.

⭐ 请 大声 读课文。

🔊 Please read the text aloud.

⭐ 他 早早 睡觉。

😴 He goes to bed early.

➕ Adding "地" (de)

✔ You can add 地 after the repeated adjective: 好好地看, 慢慢地走 (more formal/written)
✔ In spoken Chinese, it's very common to omit 地: 好好看, 慢慢走 — both are correct ✅

❌ 我好学习。
(I good study.)
✅ 我好好学习。
I study carefully.
❌ 学习好好。
(study good-good)
✅ 好好学习。
Put AA before the verb.
❌ 我很好好学习。
(Adding 很 is wrong)
✅ 我好好学习。
No need for 很.
📖 Summary (English):
• Repeat an adjective → put it BEFORE the verb → add 地 (optional).
• It shows the manner, attitude, or way of doing an action.
• Example: 好好 = “carefully/properly”; 慢慢 = “slowly”; 早早 = “early”.

📝 Board Notes

✎ Structure:

Subject + Reduplicated Adj + (地) + Verb + Object

Example: 我 好好 地 学习 汉语 ✔️ I study Chinese carefully.

🎯 When do we use it?
- Be serious / careful → 好好 prepare
- Gentle action → 轻轻 put down
- Slow / fast speed → 慢慢 run / 快快 do
- Do something early → 早 get up

🚫 Don't forget: The adjective MUST be repeated!
✅ Correct: 你慢慢说 (You speak slowly)
❌ Wrong: 你慢说

➕ 地 is optional → 好好(地)练习 practice hard
